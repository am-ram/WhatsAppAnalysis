# WhatsAppAnalysis
This is a WebApp that aims at providing insights on the WhatsApp chats. Through the expansion of WhatsApp to WhatsApp Business it has caused a rise in the number of online stores run by individuals drastically. Using this WebApp would allow them to get deep insights on their businesses and interactions with clients.
This WebApp can be used for group chats as well as individual chats. This WebApp was created using the help of streamlit and deployed to Heroku using Flask.<br><br>

## How to Upload the chat ?
1) Open the individual or group chat.
2) Tap options > More > Export chat.
3) Choose export without media.
